Fileicon.css

The Customizable pure CSS file icons

How to use GitHub

Usage

Download the fileicon.css and include fileicon.css in your website or Web app <head>. Add an element with the file-icon class. 

<div class="file-icon" data-type="doc"></div>
<div class="file-icon file-icon-lg" data-type="doc"></div>
